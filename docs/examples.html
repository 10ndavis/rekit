Examples

Examples

Below are real-world apps created with Rekit, you can look at them for reference:

  1. Rekit portal: the Rekit dev tool shipped with Rekit 2.0, it's built with Rekit so it's also a good example.
  2. Command pad: a handy GUI tool for managing command line tasks, built with Electron and Rekit 1.0.
  3. Simple forum: a simple forum application built with Rekit 1.0. It connects to a real backend server powered by Leancloud.cn( a Parse like Saas service).
